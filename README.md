趁这几天有空，就继续完善了整个系统内核的注释，每个函数以及内部实现都加了自认为相当详细的注释。相信无论对于使用还是学习来说都够用了。当然也欢迎同样有兴趣的朋友一起讨论~

写这个操作系统抱着学习的态度，主要是了解最基本的实时操作系统的工作原理，当然作为一个嵌入式工程师，自己写出来后无论如何还是会希望能应用于实际的项目中，更希望该系统能帮到同样需求简易小型操作系统的朋友。所以才会希望有相对完善的注释以及手册。

鉴于此，不出意料的话今天应该会在github上发布整理现有的注释发一个V0.1版本测试版本。使用测试版本也是因为该系统目前没用经过任何一个实际项目的检验，我会在之后自己的项目中使用该系统，逐渐修改其中bug，当有一个项目实际上市并销售之后，会把当时的版本整理并发出V1.0稳定版本。当系统在STM8S稳定之后，会考虑进一步增加可移植性，并推出STM32版本 和Arduino版本。
